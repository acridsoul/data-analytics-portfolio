## Projects

### 1. Netflix Data Wrangling
**Focus**: Data Cleaning and Transformation

A complete data wrangling workflow on the Netflix Movies and TV Shows dataset, demonstrating skills in handling missing values, data structuring, and preparation for analysis.

**Key Tasks**:
- Data loading and exploration
- Missing value treatment and duplicate removal
- Date parsing and feature extraction
- Data validation and quality checks
- Dataset export for further analysis

**Dataset**: 8,807 Netflix titles
**Output**: Cleaned dataset with 8,810 rows

### 2. Titanic Exploratory Data Analysis
**Focus**: Statistical Analysis and Visualization

Comprehensive exploratory data analysis on the Titanic dataset to uncover survival patterns and relationships between passenger attributes.

**Key Tasks**:
- Univariate, bivariate, and multivariate analysis
- Outlier detection and treatment using IQR
- Correlation analysis and feature relationships
- Target variable analysis (survival factors)
- Statistical visualizations with matplotlib and seaborn

**Dataset**: 891 Titanic passengers
**Key Finding**: Gender and class were crucial factors in survival (38.38% overall survival rate)

[View Project →](./titanic-eda)
